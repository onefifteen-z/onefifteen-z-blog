`ls` 命令是 linux 下常用命令。ls 命令就是 list 的缩写，用以查看 linux 文件夹包含的文件内容和文件信息。

<!-- more -->

### 命令格式
```
ls [选项] [目录名]
```

### 命令目标

列出目标目录中所有的子目录和文件。

### 命令参数

|命令参数|功能|
|:--|:--|
|-a|列出目录下的所有文件，包含[.a]开头的隐藏文件。|
|-A|同 `-a`，但不显示 [.当前文件夹] 和 [..上层文件夹] 。|
|-d|文件夹下自身信息表示|
|-F|文件的种类表示（`/`文件夹、`*`执行文件、`@` symbol link）|
|-i|node 号显示|
|-l|显示文件的详细信息|
|-t|按照时间顺序显示|
|-h|指定单位|

### 使用实例
#### 显示 `/home/lpic` 下文件
```
$ ls /home/lpic
sample1.txt
sample2.txt
```
